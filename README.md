# DockerOnVim

This repo is to create development environment to vim-airline on docker

# How to use

1. `docker build ./ -t vim`
2. `docker run -it -d --name vim-airline vim`
3. `docker exec -it vim-airline /bin/bash`
4. `sh ~/.vim/bin/vim_build.sh`
