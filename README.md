# NeovimOnDocker

This repo is to create development environment to vim-airline on docker

# How to use first

1. `docker build ./ -t neovim`
2. `docker run -it -d --name neovim neovim`
3. `docker exec -it neovim /bin/bash`
4. `sh ~/.config/nvim/bin/linux_build.sh`
